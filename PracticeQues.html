A span element is used for inline element.

Que2: Difference between class and IDs

Class : The class is attribute. We can use one or more class names for an HTML element. The class attribute can be used on any HTML element. We can use same class name to the differnet elements. Id attribute is written using the "." symbol.

ID : The id is attribute and it is a unique identifier. Id attribute is written using the "#" symbol.
